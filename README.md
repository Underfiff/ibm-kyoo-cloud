# Kyoo Cloud - Static Landing Page

## Description
Kyoo Cloud adalah landing page statis untuk layanan hosting server yang cepat dan aman.  
Dibuat menggunakan HTML + TailwindCSS, dapat di-deploy dengan mudah via Netlify.

## Technologies Used
- HTML5
- Tailwind CSS (CDN)

## Features
- Responsive layout
- Static hero section
- Pricing table
- FAQ section
- Mobile-friendly
